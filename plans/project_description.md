# Проект: PICASSO — Салон эстетики

## Обзор проекта
PICASSO — это современный веб-сайт премиум салона красоты полного цикла, разработанный с использованием React и современных веб-технологий. Проект представляет собой интерактивную платформу для демонстрации услуг салона, онлайн-записи и взаимодействия с клиентами через AI-консьержа.

## Основные характеристики

### 1. **Целевое назначение**
- Презентация услуг салона красоты (парикмахерские услуги, ногтевой сервис, брови, шугаринг, уход за лицом)
- Онлайн-запись на услуги
- Демонстрация работ и портфолио
- Предоставление информации о команде и ценах
- AI-консьерж для консультаций 24/7

### 2. **Технологический стек**
- **Фронтенд**: React 19.2.4 + Vite
- **Стилизация**: Tailwind CSS 4.2.2
- **Анимации**: Framer Motion 12.38.0
- **Прокрутка**: Lenis (плавная скролл-анимация)
- **Карусели**: Embla Carousel React 8.6.0
- **Маршрутизация**: React Router DOM 7.14.1
- **Иконки**: Lucide React 1.8.0
- **Бэкенд API**: Node.js (Vite proxy) + OpenRouter AI

### 4. **Ключевые функциональные возможности**

#### AI-Консьерж (Цифровой помощник)
- Интеграция с OpenRouter API (GPT-4o-mini)
- Чат-виджет в реальном времени
- Контекстная система промптов для салона красоты
- Обработка запросов на русском языке
- Автоматические ответы о услугах и ценах

#### Интерактивные элементы
- **TiltHeading**: 3D-наклон заголовков при наведении
- **MagneticButton**: Кнопки с магнитным эффектом
- **FadeIn**: Анимации появления элементов
- **TiltGlare**: Эффекты бликов и свечения
- **CanvasParticles**: Частицы на фоне героя-секции
- **Плавная прокрутка** с параллакс-эффектами

#### Адаптивный дизайн
- Полностью адаптивная верстка для мобильных устройств
- Оптимизированные изображения в формате WebP
- Темная цветовая схема премиум-класса
- Типографика с кастомными шрифтами

### 5. **Цветовая палитра**
- **GOLD (#C9A87A)**: Акцентный золотой цвет
- **TEXT (#F0EBE3)**: Основной текст
- **BG (#0E0C0B)**: Фоновый черный
- **CHOCOLATE (#151210)**: Темно-шоколадный
- **SURFACE (#1A1714)**: Поверхности элементов

### 6. **Секции сайта**
1. **Hero-секция** - Главный экран с анимациями и CTA
2. **О салоне** - Информация о философии и преимуществах
3. **Услуги** - Детализация всех предоставляемых услуг
4. **Галерея** - Портфолио работ (карусель изображений)
5. **Команда** - Профили мастеров с фото
6. **FAQ** - Часто задаваемые вопросы
7. **Контакты** - Карта, адрес, часы работы
8. **Форма записи** - Онлайн-бронирование

### 7. **Интеграции и API**
- **OpenRouter AI**: Для AI-консьержа
- **Vite Proxy**: Для обработки API запросов
- **Environment Variables**: Для хранения API ключей

### 8. **Особенности разработки**
- Использование React.lazy для code splitting
- Кастомный ErrorBoundary для обработки ошибок
- Оптимизация производительности через useCallback и useMemo
- Семантическая HTML разметка
- Доступность (ARIA атрибуты)

### 9. **Скрипты и сборка**
- `npm run dev` - Запуск dev сервера
- `npm run build` - Production сборка
- `npm run preview` - Предпросмотр production сборки
- `npm run lint` - Проверка кода ESLint
